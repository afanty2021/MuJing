# 幕境项目端到端测试计划

## Context

幕境（MuJing）是一款通过真实语境学习英语单词的桌面应用。本次端到端测试旨在全面验证应用的核心功能和用户流程，确保所有功能模块正常工作，为用户提供可靠的学习体验。

### 测试目标
1. 验证所有核心功能模块的正常运行
2. 检查功能模块间的集成和交互
3. 发现潜在的回归问题和集成缺陷
4. 验证跨平台兼容性（macOS）
5. 评估用户体验和性能表现

### 测试范围
- **功能测试**：所有主要功能模块
- **集成测试**：模块间交互和数据流
- **性能测试**：启动时间、响应速度、内存使用
- **兼容性测试**：词典加载、媒体文件播放、TTS 功能

## 测试环境

### 系统环境
- **操作系统**：macOS（Darwin 25.4.0）
- **Java 版本**：JDK 21+
- **内存配置**：-Xms64m -Xmx1g
- **垃圾回收器**：ZGC with Generational

### 测试数据准备
1. **测试视频文件**：MP4 + SRT 字幕
2. **测试文档**：TXT 文件
3. **测试词库**：JSON 格式词库文件
4. **词典数据库**：ecdict.db（需解压）
