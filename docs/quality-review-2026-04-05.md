## 五、性能审查

### 5.1 性能瓶颈

| 问题 | 影响 | 建议 |
|------|------|------|
| 大量词库数据全部加载到内存 | 内存占用高 | 实现分页或懒加载 |
| 复杂 Compose 组件渲染 | UI 卡顿 | 优化重组逻辑 |
| 弹幕系统 Canvas 渲染 | 大量弹幕时性能下降 | 限制弹幕数量 |
| SQL 查询缺少索引 | 查询慢 | 添加适当索引 |

### 5.2 内存管理

- 某些 Composable 组件没有正确处理生命周期
- 全局变量可能导致内存泄漏
- 频繁创建临时对象可能导致 GC 压力
- 已配置 ZGC 垃圾回收器（`-XX:+UseZGC`），是正确的选择

### 5.3 资源泄漏风险

- FFmpeg 进程没有明确的清理机制
- VLC 媒体播放器资源可能没有正确释放
- 某些文件流操作没有在 finally 块中关闭
